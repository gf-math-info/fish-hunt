
EXÉCUTION DE FISH-HUNT:
    1. Ouvrir le terminal
    2. Aller à la racine de fish-hunt
    3. Exécuter la commande run de gradle.
           Sur GNU/Linux : ./gradlew run
           Sur Windows   : ./gradlew.bat run


EXÉCUTION DE FISH-HUNT-SERVER:
    1. Ouvrir un autre terminal.
    2. Aller à la racine de fish-hunt-server
    3. Exécuter la commande suivante (facultatif)
           $ javac FishHuntServer.java
    4. Exécuter la commande suivante
           $ java FishHuntServer


EXÉCUTION DE FISH-HUNT-ANDROID via ANDROID-STUDIO (AVEC ÉMULATEUR)
    0. Lire le fichier Android.txt
    1. Télécharger un émulateur dans Android Studio
    2. Ouvrir le projet fish-hunt-android
    3. Exécuter le projet. 

EXÉCUTION DE FISH-HUNT-ANDROID via .APK SUR APPAREIL MOBILE
    0. Lire le fichier Android.txt
    1. Copier le fichier app-release.apk qui est situé dans le dossier 
       app/release/ à partir de la racine du projet.
    2. Coller ce fichier vers l'appareil Android. Le dossier destination 
       doit être accessible pour l'utilisateur de l'appareil.  
    3. À partir de l'appareil, ouvrir les paramètres.
    4. Aller dans Applications. 
    5. Appuyer sur le bouton menu.
    6. Appuyer sur accès spécial.
    7. Choisir Installation applis inconnues. 
    8. Autoriser Gestionnaire de fichiers à installer des applications.
    9. Ouvrir le gestionnaire de fichiers
    10. Trouver et sélectionner le fichier .apk
    11. Autoriser l'installation de l'application
    12. À l'alerte "voulez-vous vraiment intaller", sélectionner Oui.
