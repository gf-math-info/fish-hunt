
EXÉCUTION DE FISH-HUNT:
    1. Ouvrir le projet fish-hunt.
    2. Ouvrir le terminal IntelliJ (fonctionne aussi
       avec d'autres consoles).
    3. Exécuter la commande run de gradle.
           Sur GNU/Linux : ./gradlew run
           Sur Windows   : ./gradlew.bat run


EXÉCUTION DE FISH-HUNT-SERVER:
    Mêmes étapes que pour fish-hunt, mais cette fois dans le
    projet fish-hunt-server.
